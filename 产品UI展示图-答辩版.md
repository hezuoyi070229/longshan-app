# 龙山青年共创实验室 - 产品UI展示图（答辩版）

---

## 📱 一、整体设计风格

### 设计规范
- **主色调**：科技蓝 `#1890ff` + 活力橙 `#ff6b35`
- **辅助色**：成功绿 `#52c41a`、警告黄 `#faad14`、错误红 `#f5222d`
- **字体**：系统默认字体，中文优先使用微软雅黑
- **布局**：响应式设计，适配PC端和移动端
- **风格**：简洁现代、年轻化、学术氛围

## 📸 九、截图获取指南

### 如何获取真实截图

#### 方法1：本地运行截图（推荐）
1. 启动后端服务：`java -jar backend/target/*.jar`
2. 启动前端服务：`cd frontend && npm run dev`
3. 浏览器访问：`http://localhost:5173`
4. 使用浏览器开发者工具（F12）
5. 选择设备模拟器（iPhone/Android）
6. 按 `Ctrl+Shift+P` → 输入 "screenshot" → 选择 "Capture full size screenshot"

#### 方法2：使用在线工具生成 mockup
- **Smartmockups**: https://smartmockups.com/
- **Mockup World**: https://mockupworld.co/
- **Placeit**: https://placeit.net/
- **Canva**: https://www.canva.com/

#### 方法3：PPT 制作展示图
1. 使用本文档的布局描述
2. 在 PPT 中绘制线框图
3. 添加颜色和图标
4. 导出为图片
